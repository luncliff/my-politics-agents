## 시도한 것
- 축별 병합본, 개별 official fact brief, 교차 브리프를 읽어 fact ID별 공통 필드를 재정렬했다.
- 일부 공식 URL의 문서명을 web fetch로 다시 확인해 source log 명칭을 보정했다.
- 최종 통합 매트릭스와 source log를 새 파일로 생성했다.

## 성공한 것
- `final-fact-matrix.md`에 5개 섹션, 공통 컬럼, 핵심 미확인 목록을 facts only 형식으로 저장했다.
- `source-log.md`에 fact ID별 공식 문서명·기관·URL·출처 파일을 정리했다.
- 공식 문서 제목 기준으로 사업명을 통일하고 판정 규칙을 머리말에 명시했다.

## 막힌 것 / 다음에 해결
- `T-06`, `T-07`, `F-04`, `S-05`, `D-03`, `D-04`는 공식 원문 URL 또는 핵심 필드가 비어 있어 미확인으로 유지했다.
- 일부 항목은 병합본에만 예산 단서가 있고 원문 URL이 회수되지 않아 최종 판정을 보수적으로 조정했다.
- 성남시의회 원문·예산서 직접 링크가 확보되면 운영주체·유지관리 책임 공백을 추가 축소할 수 있다.
