# 경기도 참고자료

문서 갱신일: 2026-05-01

이 문서는 성남시 조사에 함께 쓰이는 경기도 단위 참고자료만 모아 둔 카탈로그입니다.

## 1. 자치법규·의정

- [행정안전부 자치법규정보시스템](https://www.elis.go.kr/locgovalr/locgovSeAlrList): 경기도 자치법규 검색 페이지
- [경기도](https://www.gg.go.kr/contents/contents.do?ciIdx=675&menuId=2454): 경기도 자치법규
- [경기도의회](https://www.ggc.go.kr/): 의정정보 메인 포털
- [경기도의회](https://kms.ggc.go.kr/svc/cms/mnts/MntsLatelyList.do): 최근 회의록 목록
- [경기도의회](https://www.ggc.go.kr/site/agendaif/home): 의안정보
- [경기도의회](https://www.ggc.go.kr/site/main/committee/stndnCmt?cmtCd=01): 상임위원회
- [경기도의회](https://www.ggc.go.kr/site/main/board/clnc_rsgrp/list): 의원연구단체

## 2. 통계·재정·데이터

- [경기도청](https://stat.gg.go.kr/statgg/main.html): 경기통계 및 시군 통계 검색
- [경기도](https://data.gg.go.kr/portal/mainPage.do): 경기데이터드림
- [경기도여성가족재단](https://www.gwff.kr/base/contents/view?contentsNo=78&menuLevel=3&menuNo=32): 여성·가족 분야 통계 아카이브

## 3. 연구기관·정책자료

- [경기연구원](https://www.gri.re.kr/web/contents/resreport.do?schM=list&viewCount=100): 연구보고서 목록
- [경기도교육연구원](https://www.gie.re.kr/edudata/basic/basicStatistics.do): 경기교육 기본통계
- [경기도교육연구원](https://www.gie.re.kr/edudata/policy/policyResearch.do): 교육 정책연구 데이터 및 관련 연구 주제 열람
- [경기도여성가족재단](https://www.gwff.kr/): 복지·가족 정책기관 메인 포털
- [경기도여성가족재단](https://www.gwff.kr/base/board/list?boardManagementNo=1&menuLevel=3&menuNo=26): 정책연구 목록

## 4. 청년·교육

- [경기도](https://youth.gg.go.kr/gg/archive-policy-search.do): 경기청년포털 정책 아카이브 검색
- [경기도교육청](https://www.goe.go.kr/): 교육정책 및 행정정보 공식 포털

## 5. 사용 메모

- 경기도 단위 제도, 통계, 의정자료를 먼저 찾을 때 이 문서를 사용합니다.
- 성남시 고유 자료는 `docs/references-성남시.md`를 우선 봅니다.
- 중앙정부·전국 공통 진입점은 `docs/references.md`에서 요약을 확인합니다.
