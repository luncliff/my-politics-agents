# 거버넌스

이 저장소가 다루는 자료의 상당수는 **공공데이터**이지만, 인용·재배포·가공에는
법적·윤리적 책임이 따릅니다. 이 문서는 우리 모두가 지키는 최소 규칙을 정의합니다.

## 1. 출처(Citation) 규칙

- 외부 자료에서 파생된 모든 산출물은 다음을 포함합니다.
  - 원본 URL
  - 수집 일시 (ISO-8601, KST)
  - 본문 SHA-256 해시 (앞 12자만 표기 가능)
- 인용 메타는 `gov-archive` MCP의 `archive_cite` 도구로 생성합니다.
- 부분 발췌는 `>` 인용 블록과 출처 링크를 함께 둡니다.

## 2. 라이선스

| 자료 | 일반적 라이선스 | 비고 |
| --- | --- | --- |
| 법령 본문 (`law.go.kr`) | 공공저작물 / 자유이용 | 변형·해설 가능 |
| 의안·회의록 (국회·지방의회) | 대체로 공공누리 1유형 | 사이트별 확인 필수 |
| 행정기관 공고·고시 | 공공누리 1~4유형 혼재 | 4유형은 비상업 한정 |
| 시민 의견(블로그·SNS) | 작성자 저작권 | 직접 인용 시 동의 필요 |

확실치 않으면 **재배포 금지**로 간주하고 요약·분석만 게재합니다.

## 3. PII(개인정보)

- 회의록의 민원인 이름·전화번호·주소는 `pii-mask` 스킬로 마스킹한 뒤에만 저장·공유합니다.
- 마스킹 매핑 키는 디스크에 저장하지 않습니다(메모리 한정).
- 원본은 `archive/raw/`에 보존하되, **개인이 직접 열람**하는 용도로만 사용합니다(공유 금지).

## 4. 정치적 중립

- 정당·후보·특정 인물에 대한 산출물은 다음 두 영역을 분리합니다.
  - **사실 카드(Facts)** — 검증 가능한 1차 자료 인용
  - **해석(Interpretation)** — 작성자/에이전트의 추론·평가
- 사실 카드에는 출처 링크가 100% 포함되어야 합니다.

## 5. 자동화 윤리

- `robots.txt` 위반 금지.
- 동일 호스트 1초 1회 이상 요청 금지(어댑터에서 강제).
- 사이트 ToS에 "자동 수집 금지"가 있다면 수집 중단, 사람의 수동 처리 권장.

## 6. 기여 가이드

- 새로운 사이트 어댑터 PR은 다음을 포함합니다.
  - 대상 사이트의 ToS·robots.txt 요약
  - 샘플 데이터 1건과 인용 메타
  - PII 발생 가능성 평가
- 회고(`retrospectives/`)가 새 PR의 근거가 됩니다.

## 7. 분쟁·삭제 요청

- 산출물 삭제 요청을 받으면 24시간 이내에 처리하고, `archive/raw/`의 해당 항목도
  접근 제한합니다(완전 삭제 여부는 사안별 판단).
- 처리 결과는 `retrospectives/`에 사실만 기록합니다.
