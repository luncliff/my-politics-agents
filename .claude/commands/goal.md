# 세션 목표 설정 및 추적

세션 목표를 선언하거나, 기존 목표의 진행 상태를 확인한다.

## 입력

- `$ARGUMENTS`: 목표 설명 (자연어). 생략 시 잔여 목표 표시.

## 절차

1. 스킬 [track-goals](../../.agents/skills/track-goals/SKILL.md)를 따른다.
2. `.goals/current.md`가 없으면 생성한다.
3. 인자가 있으면 새 목표를 등록하고 하위 단계를 분해한다.
4. 인자가 없으면 현재 목표와 진행 상태를 표시한다.
5. 목표 등록 후 첫 실행 단계를 사용자에게 확인한다.

## 출력

- 목표 + 단계 목록을 표 또는 체크리스트로 표시.
- 각 단계에 매핑된 agent/skill 명시.
