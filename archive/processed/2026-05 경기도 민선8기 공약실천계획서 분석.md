# 경기도 민선8기 공약실천계획서 분석

기준 2026-05, 대표 출처는 경기도지사 공식 누리집 공약사항 페이지와 `민선8기 공약실천계획서(22.12.29. 기준)` PDF입니다.

## 입증됨

- 공식 원문은 경기도지사 공식 사이트 `도민과의 약속 > 공약사항`에서 제공되며, PDF 제목은 `민선8기 공약실천계획서(22.12.29. 기준)`입니다.
- 문서는 `3대 비전`, `9대 분야`, `295개 실천과제` 체계로 편성되어 있습니다.
- 총론에는 `공약 총괄`, `공약 예산`, `공약 목록`이 먼저 나오고, 이후 각 실천과제가 동일한 카드 형식으로 반복됩니다.
- 확인한 상세 과제 1번과 2번 모두 아래 공통 필드를 같은 순서로 사용합니다: `임기구분`, `사업주체`, `신규여부`, `사업시기`, `예산구분`, `추진현황`, `정책목표`, `주요현황`, `정책개요`, `추진계획`, `소요예산`, `공약달성 확인지표`, `기대효과`, `담당부서·연락처`.
- `추진계획`은 연도와 분기 단위로 적는 간트형 표이고, `공약달성 확인지표`와 `소요예산`은 각각 별도 표로 분리됩니다.

## 구조 분석

### 1. 문서 상단 총론

- 표지: 문서명만 크게 제시
- 총괄: 비전 문구와 전체 공약 체계 설명
- 예산 총괄: 전체 재원 규모 제시
- 공약 목록: 분야별 실천과제 번호, 과제명, 담당 부서, 페이지

### 2. 본문 편성 단위

- 대단락: `Ⅰ. 더 많은 기회`, `Ⅱ. 더 고른 기회`, `Ⅲ. 더 나은 기회`
- 중단락: 9개 분야
- 세부 단위: 1번부터 295번까지 각 실천과제

### 3. 과제별 카드의 공통 골격

| 구간 | 필수 항목 | 용도 |
|---|---|---|
| 기본정보 | 과제번호, 과제명, 임기구분, 사업주체, 신규여부, 사업시기, 예산구분, 추진현황 | 과제 분류와 상태 표시 |
| 배경 | 정책목표, 주요현황 | 과제 필요성과 현행 조건 설명 |
| 실행설계 | 정책개요, 추진계획 | 실행 방식과 일정 제시 |
| 성과관리 | 공약달성 확인지표 | 정량 목표와 연도별 목표치 관리 |
| 재정관리 | 소요예산 | 총액, 재원별 구분, 연도별 예산 배치 |
| 기대값 | 기대효과 | 정책효과 설명 |
| 책임정보 | 담당부서, 팀장, 담당자, 전화번호 | 사후 관리와 문의 대응 |

### 4. 표 설계 특징

- `추진계획`은 `2022~2026` 연도와 `1~4분기` 열을 사용합니다.
- `공약달성 확인지표`는 `확인지표`, `단위`, `목표`, `총계`, `연도별 목표`를 함께 둡니다.
- `소요예산`은 `계`, `국비`, `도비`, `시군비`, `기타` 같은 재원 구분을 쓰고, 총액과 연도별 금액을 함께 둡니다.

## 실무 해석

1. 이 문서는 단순 공약 목록이 아니라 `과제 카드 + 성과표 + 예산표`를 결합한 집행관리 문서입니다.
2. 따라서 양식은 `총론 양식`과 `개별 과제 양식`을 분리해야 재사용성이 높습니다.
3. 정량관리의 핵심은 `연도별 목표`와 `연도별 예산`이므로, 양식에서도 이 두 표를 생략하면 원문 구조를 재현하지 못합니다.
4. 담당부서와 연락처 줄은 공식 문서 성격상 중요하므로 별도 블록으로 유지하는 편이 맞습니다.

## 생성 양식 적용 원칙

- 문서 1권 기준으로 `표지 > 총괄 > 분야별 목록 > 개별 과제 시트 반복` 순서를 유지합니다.
- 개별 과제 시트는 한 과제당 독립적으로 복제할 수 있게 설계합니다.
- 예산이 없는 과제도 존재할 수 있으므로 `예산구분`과 `소요예산` 표는 `비예산` 입력을 허용해야 합니다.
- 시군 또는 타 광역단체에 전용할 수 있도록 기관명은 일반화하되, 경기도 원문 필드 순서는 보존합니다.
