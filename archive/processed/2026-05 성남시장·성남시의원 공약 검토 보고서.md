# 성남시장·시의원 후보 공약 검토 보고서

기준: 2026-05 · 대상 선거구: 성남시 분당구 자 (판교동·운중동·백현동·대장동)

분석 차원 4가지: ① 정책 방향 일치 여부, ② 시의원 공약이 시장 공약의 지역 실행과제로 기능하는지, ③ 시청 외부 협의 기관 수, ④ 권한 분담 구조 (성남시청–성남시의회–경기도청·도의회–경기도교육청–중앙부처)

## A. 공약 정렬 현황

### A-1. 직접 정렬 항목

시장·시의원 공약 문구가 동일 사업을 지칭하며, 시의원 공약이 세부 실행수단을 추가한 항목.

| 시장 후보 공약 | 시의원 후보 공약 | 정렬 유형 |
|---|---|---|
| 판교동 578번지 기업 유치 | 578번지 기업 유치 + 복합문화체육시설 조기 착수 | 직접 — 부지 활용 방식 구체화 |
| KAIST 김재철 AI 대학원 조기 준공 | 조기 준공 + 주민 참여형 프로그램 도입 | 직접 — 지역 환원 추가 |
| 운중천 정비 연계 운중저수지 수변공원 | 수변공원 + 수변산책로 조성 | 직접 — 하천·산책로·공원 세분화 |
| 판교대장 초·중 과밀 해소 + 고교 설립 | 학교 과밀 공동대책·급식실·체육관 증축·고교 설립 | 직접 — 단기·중기 과제 분리 |
| 판교대장복지관 내 청소년 전용시설 | 종합사회복지관·도서관 조기 준공 + 청소년 전용시설 | 직접 — 선행 인프라 연결 |
| 백현 MICE 조기 착공 + 마이스역 신설 | 동일 | 직접 — 문구 일치. 행정 단계는 초기. |
| 물방울공원 공영주차장 재추진 | 동일 | 직접 — 생활권 주차 현안 |

### A-2. 보완 정렬 항목

시장 공약에 직접 문구는 없으나, 상위 정책 방향의 실행수단으로 기능.

- 백현동 더샵판교퍼스트파크–판교동 접근성 개선 · GTX-A 성남역 환승불편 해소 · 백현MICE–판교동 연결도로 및 보행동선 확보 → 시장 공약의 백현 MICE·생활권 개발을 이동 문제로 구체화
- 판교초 특수학급 시-교육청 공동지원 · 학부모 협의체 · 판교형 공공 방과후 캠퍼스 · 판교-삼평 AI·IT 진로교육 연합 → 판교권 교육 기반 강화 방향과 병행
- 운중동 복합문화체육센터 · 운중동 공영주차장 · 대장동 어린이집 확충 → 대장·운중 생활SOC 축 보완
- 판교동역 신설 · 판교동 공영주차장 → 판교동 교통 인프라 보완
- 대장동 소방서·파출소 신설 → 대장지구 안전 인프라 보완

### A-3. 시의원 독자 공약 (시장 공약 대응 없음, 충돌 없음)

판교 상권을 잇(IT)다 · 테마거리 · 전자스탬프 투어 · 동네 축제 · 동아리 사업 부활 → 생활권 활성화 별도 축
