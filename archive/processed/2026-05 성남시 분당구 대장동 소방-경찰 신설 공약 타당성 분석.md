# 성남시 분당구 대장동 소방서·파출소 신설 공약 타당성 분석

## 1. 대장동 현황 요약

### 1-1. 지역 개요
- **위치**: 성남시 분당구 북동부 (판교테크노밸리 인접)
- **개발사업**: 판교대장지구 도시개발
- **규모**: 약 2만호 예상, 인구 5만명 이상 예상
- **입주시기**: 2026~2028년 본격화

### 1-2. 현재 응급서비스 커버리지

#### 소방 현황
| 구분 | 현황 |
|------|------|
| 관할 소방서 | 분당소방서 (수내동 소재) |
| 분당소방서~대장동 거리 | 약 12~15km (자동차 기준 약 15~20분) |
| 현지 119안전센터 | 없음. 향후 대장동 내 설치 예정 |
| 응답목표 | 119 신고 후 5분 이내 현장 도착(황금시간) |
| 신도시 진입로 | 신분당선, 버스 중심 (소방차 접근성 양호) |

#### 경찰 현황
| 구분 | 현황 |
|------|------|
| 관할 경찰서 | 분당경찰서 (금곡동 소재) |
| 관할 지구대 | 미정 (관할구역 확인 필요) |
| 분당경찰서~대장동 거리 | 약 8~12km |
| 신설 가능성 | 신도시 규모·주민 민원 고려 시 중상 |

## 2. 권한 구조 (소방·경찰)

### 2-1. 소방서 신설 권한

#### 법적 근거
- **소방기본법** (법률 제20104호)
  - 제2조: 중앙·광역·기초 소방조직 구성
  - 제3조~5조: 소방청, 소방본부, 소방서·119안전센터 설립의 기본 방향

- **지방소방기관설치에관한규정** (대통령령)
  - 소방서 신설 기준: 인구 기준, 면적 기준, 응답시간 기준
  - 권한: 소방청(중앙)이 광역 계획 수립, 경기도 소방본부가 기초자치단체 범위 내 운영

#### 결정 권한 소재
```
소방청(중앙)
  ↓
경기도 소방본부(광역)
  ↓
기초자치단체 협의 (성남시)
  ↓
[최종 승인: 경기도 소방본부]
```

- **최종 결정권**: 경기도 소방본부 (소방청 상위 승인하)
- **성남시 역할**: 부지 제공, 건축비 매칭, 인사 지원 (결정권 X)

#### 설치 기준
| 기준 | 내용 |
|------|------|
| 인구 기준 | 기초자치단체당 약 10만 명 당 1개 소방서 |
| 면적 기준 | 관할면적 30km² 이상 시 신설 검토 |
| 응답시간 | 119 신고 후 5분 이내 현장 도착(황금시간 원칙) |
| 신도시 특칙 | 입주 인구 10만명 이상 신도시는 소방서 또는 119안전센터 조기 설치 |

#### 대장동의 소방서 신설 가능성 평가

**유리한 요소**:
- 신도시 규모 (2만호 = 5만 인구) + 기존 분당 인구 집중 → 소방수요 증가 명백
- 응답시간: 분당소방서에서 15~20분 → 황금시간 원칙에 미치지 못할 수 있음
- 현지 119안전센터 설치는 확정단계 (소방서 신설은 그 이후 검토 가능)

**제약 요소**:
- 경기도 재정 제약 (신도시 소방서는 '신설'이 아니라 '신청' 단계가 오래걸림)
- 분당소방서가 2018년 신설되었으므로, 3년 이내 재신설 결정 어려움
- 소방공무원 정원 제약 (전국 소방청 정원 한도 내에서만 신설)

**추정 타이밍**:
- **119안전센터**: 2027~2028년 개설 가능 (신도시 본격 입주 시기)
- **소방서 분소 또는 신설**: 2028~2030년 이후 검토 가능

### 2-2. 파출소·지구대 신설 권한

#### 법적 근거
- **경찰법** (법률 제21086호)
  - 제2조: 자치경찰·국가경찰 조직
  - 제11조: 지구대·파출소 설치 권한

- **경찰청과그소속기관직제** (대통령령)
  - 경기남부경찰청이 경기도 남부 경찰 업무 담당
  - 지구대·파출소 신설은 경찰청 승인

- **국가경찰과자치경찰의조직및운영에관한법률** (2024년 개정)
  - 2024년 상반기 자치경찰 전환으로 지방 조직 재개편

#### 결정 권한 소재
```
경찰청(중앙, 국가경찰 담당)
  ↓
경기남부경찰청
  ↓
지구대/파출소 신설 검토 (시군 요청 기반)
  ↓
[최종 승인: 경기남부경찰청]
```

- **최종 결정권**: 경기남부경찰청 (경찰청 감시 하)
- **성남시**: 건의안 제출, 주민 민원 취합 가능 (결정권 X)
- **자치경찰 역할**: 2024년 지방치안 강화 조항으로 성남시 자치경찰팀이 참여 가능

#### 설치 기준
| 기준 | 내용 |
|------|------|
| 인구 기준 | 지구별 10~15만 명 당 1개 지구대, 파출소는 주거지역별 |
| 면적 기준 | 관할면적 30km² 이상 시 파출소 추가 설치 |
| 치안수요 | 민원 발생, 범죄율, 유동인구, 신도시 지정 여부 |
| 신도시 특칙 | 입주인구 5만명 이상 신도시는 파출소 신설 검토 |

#### 대장동의 파출소 신설 가능성 평가

**유리한 요소**:
- 신도시 규모 (5만명 이상) → 자동 검토 대상
- 기초단체 신청 건의 시 경찰청이 검토하는 프로세스 확립됨 (2024~ 자치경찰 강화)
- 신도시 입주 시간차(2027~2028)를 기준으로 1~2년 전부터 신설 공사 가능

**제약 요소**:
- 경찰청 정원 문제: 전국 경찰 정원 증가는 국가예산 한도 내에서만 가능
- 경기남부경찰청 관할 지역 전체 수요 고려 (대장동만이 아니라 경남부 전체 우선순위)
- 2024년 자치경찰 전환 이후 중앙 정부 신설 승인 절차 변경 가능성

**추정 타이밍**:
- **파출소 또는 지구대 신설**: 2027~2029년 검토 가능
- **최단 시나리오**: 성남시+분당구+주민회 공동 건의 → 경기남부경찰청 긍정 검토 → 2027년 착공 → 2028년 개설

## 3. 시의원으로서의 역할과 한계

### 3-1. 시의원이 할 수 있는 것 (권한 範圍內)

| 구분 | 구체적 방법 |
|------|-----------|
| **의회 활동** | • 소방·경찰 신설에 관한 결의안 발의 및 의결<br>• 예산 편성 시 "소방·경찰 신설 협의비" 또는 "부지 조성비" 계상<br>• 행정안전부/소방청/경찰청 업무보고 시 질의 |
| **행정 협력** | • 시장/분당구청과 함께 광역(경기도 소방본부, 경기남부경찰청) 정식 건의<br>• 부지 제공 가능 여부 확인 (도시개발공사와 협의)<br>• 매칭펀드 예산 반영 (시 차원의 재정 지원) |
| **주민 조직화** | • 주민 청원/서명 수집 → 시장에 전달<br>• 주민 간담회 개최 (소방청·경찰청 담당자 초청)<br>• 신도시 입주민 수요조사 |
| **정부 건의** | • 국회의원(분당 관할)에 건의안 의뢰<br>• 경기도의원(분당 관할)과 연계 건의<br>• 경기도청 소방재정과/경기남부경찰청 방문 간담회 |

### 3-2. 시의원이 할 수 없는 것 (결정권 부재)

| 제약 | 이유 |
|------|------|
| 소방서 최종 결정 | 경기도 소방본부 권한. 시의원이 결정권 없음. |
| 경찰 파출소 결정 | 경기남부경찰청 권한. 시의원이 결정권 없음. |
| 예산 배정 보장 | 중앙 또는 광역 재정 결정. 기초자치단체(성남시)도 확보 보장 불가. |
| 우선순위 변경 | 경기남부권 전체 신설 우선순위는 경찰청이 결정. |

## 4. 공약 표현 권고

### 4-1. 문제 진단
**현재 공약 형태가 위험한 이유**:
- 소방서/파출소 "신설"은 기초자치단체(성남시)와 시의원의 결정권이 거의 없음
- 주민은 "공약 실현"을 기대하지만, 실제는 "광역 및 중앙 정부 건의"의 성격
- 임기 4년 중 신설이 이루어지지 않을 가능성 높음 → 신뢰도 손상

### 4-2. 권고 표현 (개선 안)

#### ❌ 피해야 할 표현
```
"대장동 소방서 신설 추진"
"판교대장 파출소 신설 공약"
```

#### ✓ 권고하는 표현

**표현 1 (보수적, 실행 가능성 높음)**:
```
"대장동 신도시 입주민 응급서비스 강화"
  - 119안전센터 조기 개설 추진
  - 소방·경찰 신설 필요성 광역 건의
  - 응답시간 단축을 위한 소방/경찰 협력체계 구축
```

**표현 2 (중간 수준, 분명한 목표)**:
```
"분당권 응급·치안 인프라 현대화 건의"
  - 분당소방서/분당경찰서 분소(분소) 또는 신설 추진 촉구
  - 경기도·경찰청과 신도시 인구 대응 용역 공동 수행
  - 신도시 입주 전 기본 응급서비스 기준 달성 목표
```

**표현 3 (강한 표현, 리스크 있음)**:
```
"대장동 응급·치안 대응 인프라 확보"
  - 소방 안전센터 2027년 개설
  - 파출소 신설 촉구 (2028~2030년 목표)
  - 기초자치단체-광역자치단체 협력 강화 및 정부 예산 지원 촉구
```
→ 이 경우 "추진"이 아니라 "촉구", "목표"라는 조건부 표현 필수

## 5. 신도시 개발 사례 (참고)

### 5-1. 세종시 (신행정수도)
- **규모**: 약 50만 인구
- **소방서**: 신설 2개 소방서 + 119안전센터 다수
- **경찰**: 세종경찰청 신설 (2012)
- **타이밍**: 신도시 준공 시점과 동시 (또는 1년 전 사전 배치)

### 5-2. 동탄신도시 (화성시)
- **규모**: 약 30만 인구
- **소방**: 119안전센터 먼저 배치 (2010) → 소방서 신설 (2015)
- **경찰**: 동탄 지구대 신설 (2012)
- **타이밍**: 신도시 본격 입주 2년 전부터 인프라 배치 시작

### 5-3. 위례신도시 (남양주시)
- **규모**: 약 5만 인구 (대장동과 유사)
- **소방**: 119안전센터 설치 (2023), 소방서 신설은 미정
- **경찰**: 파출소 신설 승인 (2023년 주민 건의 → 경찰청 긍정 검토 → 2024년 신설)
- **타이밍**: 주민회 결성 → 공식 건의 → 1~1.5년 후 승인

**위례 사례의 교훈**:
- "신도시 입주 전 파출소 신설 촉구" → 경찰청이 검토 (신도시 특례)
- 주민회 공식 건의가 효과적
- 시의원의 공식 건의안보다는 주민 민원이 더 실질적 영향

| 항목 | 최종 결정권 | 기초자치단체 권한 | 시의원 권한 |

| **소방서 신설** | 경기도 소방본부 (소방청 상위) | 건의, 부지 제공 협의 | 결의안 발의, 광역 건의 |

| **경찰 파출소/지구대** | 경기남부경찰청 (경찰청 상위) | 건의, 부지 협의 | 결의안 발의, 광역 건의 |
| **응급서비스 강화 협의체** | 시 차원 | **시장/분당구청 주도** | 주민 의견 수렴 |

## 7. 최종 권고

### 7-1. 공약 작성 시 조언
1. **직접 신설 약속 금지**: "신설 추진"이 아니라 "신설 촉구", "건의", "협력"으로 표현
2. **기한 명시 시 조건 달기**: "2028년 파출소 개설 추진 목표" (단, 정부 결정 전제)
3. **주민 민원 조직화 포함**: 시의원 혼자가 아니라 "주민 청원 + 시의원 의회활동" 병행 강조
4. **예산 명시 피하기**: 광역/중앙 정부 사업이므로 성남시 예산 배정 불명확

### 7-2. 실행 로드맵 (현실적 타이밍)
```
2026년
  ↓ 주민회 발족 + 공식 건의안 수집
2027년
  ↓ 시의원/시장 → 경기도/경찰청 정식 건의
  ↓ 119안전센터 개설
2028년
  ↓ 신도시 본격 입주 + 치안 수요 증가
2028-2029년
```
