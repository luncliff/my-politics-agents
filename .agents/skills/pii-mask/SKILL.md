# pii-mask

## 정책

- **마스킹 대상**
  - 주민등록번호: `\d{6}-\d{7}` → `[RRN]`
  - 휴대전화: `01[016-9]-?\d{3,4}-?\d{4}` → `[PHONE]`
  - 이메일: `[\w.+-]+@[\w-]+\.[\w.-]+` → `[EMAIL]`
  - 도로명/지번 주소 패턴 → `[ADDRESS]`
  - 일반인 이름(2~4자 한글, 직책 없는 호칭) → `[NAME]` (휴리스틱, 공직자는 보존)
- **보존**
  - 공직자(시장·의원·국장 등 직책이 명시된 인물)는 마스킹 제외 — 공적 인물.
  - 법인명은 마스킹 제외.
- **저장 정책**
  - 매핑 키(원본 ↔ 토큰)는 **메모리만**. 디스크에 저장 금지.
  - 마스킹 적용 사실을 frontmatter `pii_masked: true`로 표기.

## 절차

1. 입력 텍스트에서 위 패턴을 순서대로 치환.
2. 직책 동반 이름은 보존 규칙에 따라 다시 복원.
3. 결과 텍스트를 반환. 호출자가 저장 책임.
